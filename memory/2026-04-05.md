## 13:47 — RippleMaster / RippleMonitor 第二轮优化

### master API 故障修复
**问题：** master API (port 3002) 所有端点返回 HTTP 200 但 body 全空
**根因：** `/opt/master-api/app.py` 使用了 `import requests`，但 `/opt/mirofish/venv/` 虚拟环境中未安装 requests；flask-cors 捕获 ImportError 后静默返回空响应
**修复：** 重写 `/opt/master-api/app.py`，用 Python 标准库 `wsgiref.handlers` 替代 uvicorn/gunicorn，移除所有 `requests` 依赖；Flask 自带 wsgiref 不需要额外包
**验证：** `curl http://127.0.0.1:3002/api/master/simulate` 返回完整 JSON

### nginx `;n` 字面量故障
**问题：** 用 sed 插入 nginx proxy_set_header 时，`\n` 被 shell 展开成字面 `;n`，导致指令截断
**根因：** `sed -i 's|proxy_pass http://127.0.0.1:3002;|proxy_pass http://127.0.0.1:3002;\n    proxy_set_header X-Real-IP|'`
**修复：** 用 Python 脚本整体替换整个 location 块，彻底重写而非 sed 补丁
**教训：** nginx 配置块修改用 Python 整体替换，不要用 sed 插入行
