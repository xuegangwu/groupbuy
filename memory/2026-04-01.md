# 工业园区数字孪生平台 — 技术方案

**来源**：伍学纲提供（2026-04-01）
**版本**：v1.0

---

## 项目概述

面向工业园区场景的"源-网-荷-储"一体化数字孪生平台，整合光伏储能、充电桩、虚拟电厂三大模块。

---

## 核心技术架构

```
应用层 → VPP对接层 → 策略层 → 孪生层 → 数据层 → 采集层 → 端侧
```

| 层级 | 技术选型 |
|:---|:---|
| 3D渲染引擎 | Three.js + React Three Fiber |
| GIS支持 | Cesium.js（可选） |
| 时序数据库 | TDengine |
| 实时缓存 | Redis |
| 消息队列 | Kafka / EMQX |
| AI预测 | LSTM / Prophet |
| 优化求解 | Pyomo / ortools |

---

## 三项目定位

| 模块 | 说明 | 优先级 |
|:---|:---|:---|
| 光伏储能 | PV、BESS数据采集+调度优化 | 核心 |
| 充电桩 | EV充电管理+有序充电 | 核心 |
| 虚拟电厂 | VPP聚合+市场交易+收益分配 | 高阶 |

---

## 三维可视化功能

- 园区1:1三维可视化（Three.js）
- 设备状态实时映射（颜色/材质变化）
- 能量流动粒子可视化
- 设备点击数据面板
- VPP状态聚合面板

---

## AI调度策略

| 策略 | 算法 | 目标 |
|:---|:---|:---|
| 负荷预测 | LSTM/Prophet | MAPE < 10% |
| 光伏预测 | LSTM+物理模型 | 晴天MAPE<8% |
| 削峰填谷 | MILP/动态规划 | 电费最小化 |
| 需量管理 | 规则+实时响应 | 需量<契约值95% |
| VPP市场决策 | 博弈论 | 收益最大化 |

---

## VPP对接

| 对接方 | 协议 | 内容 |
|:---|:---|:---|
| 电网调度平台 | IEC 104/61850 | 遥测、遥控 |
| VPP聚合平台 | REST API/MQTT | 容量上报、响应确认 |
| 电力交易中心 | OAuth2 API | 行情、申报、结算 |
