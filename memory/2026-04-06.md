## 身份认证体系上线（12:20前后）

### 完成内容
- **后端 Auth API**（enos-api, port 8080）：
  - `POST /api/auth/register` — 注册（姓名/邮箱/密码/公司/电话），bcrypt加密
  - `POST /api/auth/login` — JWT → httpOnly Cookie（7天），Cookie名`token`
  - `GET /api/auth/me` — 获取当前登录用户
  - `POST /api/auth/logout` — 登出
  - authMiddleware 支持 Authorization Header 和 Cookie 两种 Token 读取方式
- **前端首页（solaripple.com）**：
  - 右上角"开始使用" → 登录弹窗
  - 弹窗内切换登录/注册两个表单
  - 登录后 Nav 显示用户名，点击可再次打开用户面板
  - 页面加载时 checkLoginState() 自动检测登录态
  - GitHub 已推送：xuegangwu/solaripple
