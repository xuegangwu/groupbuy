## InfluxDB 真实数据修复

**问题根因**：execSync 返回数据但被错误过滤

1. **逗号开头行过滤错误**：原过滤条件 `!startsWith(',')` 把所有 InfluxDB CSV 数据行都删了（InfluxDB CSV 数据行以逗号开头）
2. **字符串字段 schema collision**：`is_peak`、`period`、`region` 等字符串字段导致 group 冲突
3. **group 策略错误**：`group(station_id) |> last()` 只返回每组一个 field（被 temperature_c 占了）
4. **price 用 station_id 无法匹配**：electricity_price 没有 station_id，只有 region

**最终正确查询**：
```javascript
// 电站数据：过滤字符串字段 + group(station_id, _field)
'from(bucket: "enos") |> range(start: -24h) |> filter(fn: (r) => r._measurement == "X" and r._field != "is_peak" and r._field != "period") |> group(columns: ["station_id", "_field"]) |> last()'

// 电价：直接取 last()
'from(bucket: "enos") |> range(start: -24h) |> filter(fn: (r) => r._measurement == "electricity_price" and r._field == "price_yuan_per_kwh") |> last()'
```

**最终数据结构**：
- station-001（苏州工业园）：光伏2438kW / 负荷1812kW / SOC 87% / 电价 ¥1.051
- station-002（杭州未来科技城）：光伏1949kW / 负荷2004kW / SOC 25% / 电价 ¥1.051
- station-003（南京江北新区）：光伏1690kW / 负荷1708kW / SOC 25% / 电价 ¥1.051
