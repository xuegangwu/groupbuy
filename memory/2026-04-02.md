# 2026-04-02 日记

## 动态电价功能（对标章鱼能源）

### 完成内容

**后端 API 新增：**
- `GET /api/market/provinces` — 支持省份列表（浙江/广东/上海）
- `GET /api/market/pricing/:province` — 完整分时电价数据（24h曲线 + 最优策略）
  - 优先返回 MongoDB 真实爬取数据
  - 无真实数据时 fallback 政策参考电价
  - metadata.dataSource 标识：real / real-unverified / synthetic
  - `triggerScrape=true` 可触发实时爬取
- `GET /api/market/compare` — 浙江 vs 广东 vs 上海电价对比
- `GET /api/market/solar-overlap` — 光伏发电与电价重叠分析

**MongoDB 模型（新增）：**
- `MarketPrice` — 爬取的分时电价记录

- `ScraperLog` — 爬虫执行日志

- `MarketScraperScheduler` — 统一调度器（每日16:00）
